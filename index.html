Grammar Notation

In his book Mindstorms, Seymour Papert encourages us to find and create tools that make abstract ideas more concrete. Below is a simple grammar notation system. Each part of speech (nouns, verbs, adjectives, etc.) is given a specific colour. Type or paste whatever you want in the text box and click the button to generate the notation.

This was made for diversity p5.js Mindstorms book club, and uses p5.js and rita.js. Source code.
